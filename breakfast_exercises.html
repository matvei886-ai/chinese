1 Выберите правильный перевод

Нажмите на правильное значение иероглифа или фразы.

油条 = ?
✅ 油条 (yóutiáo) — жареные хрустящие палочки из теста, классика китайского завтрака
豆浆 = ?
✅ 豆浆 (dòujiāng) — соевое молоко. 豆 = боб/соя, 浆 = жидкость. Классическая пара с油条.
拿铁 = ?
✅ 拿铁 (nátiě) — латте. Транскрипция итальянского latte. Капучино = 卡布奇诺/卡布, эспрессо = 浓缩咖啡.
培根 = ?
✅ 培根 (péigēn) — бекон (транскрипция английского bacon). Свиная сосиска = 猪肉香肠.
暖胃 = ?
✅ 暖胃 (nuǎnwèi) — согревает желудок. 暖 = тёплый/тепло, 胃 = желудок. Говорят про горячую кашу.
水煮蛋 = ?
✅ 水煮蛋 (shuǐzhǔdàn) — варёное яйцо. 水 = вода, 煮 = варить, 蛋 = яйцо.
减脂早餐 = ?
✅ 减脂 (jiǎnzhī) = снижать жир/худеть. 减脂早餐 = завтрак для похудения: 少油少盐超健康.
低血糖 = ?
✅ 低血糖 (dī xuètáng) — гипогликемия / низкий сахар. 低 = низкий, 血 = кровь, 糖 = сахар.
鲜榨橙汁 = ?
✅ 鲜榨 (xiānzhà) = свежевыжатый. 鲜 = свежий, 榨 = выжимать. Ср. просто 橙汁 = апельсиновый сок.
边缘脆脆的 = ?
✅ 边缘 = край/кромка, 脆脆的 = хрустящий. Так описывают идеальную жареную яичницу!

3 Выберите подходящее слово

Какое слово подходит по смыслу в предложении?

不吃早饭对___不好。
Пропускать завтрак вредно для ___.
✅ 胃 (wèi) = желудок. Дословно: «Не есть завтрак для желудка нехорошо».
赶时间就做___早餐。
Когда торопишься, делай ___ завтрак.
✅ 快手 (kuàishǒu) = быстрый, ловкий. 快手早餐 = быстрый завтрак. 赶时间 = торопиться.
鸡蛋很有___。
Яйца очень ___.
✅ 营养 (yíngyǎng) = питательная ценность / питателен. 很有营养 = очень питателен/полезен.
减脂早餐少___少盐超健康。
Завтрак для похудения: мало ___, мало соли — очень полезно.
✅ 少油少盐 = мало масла, мало соли. 油 = масло/жир. Формула здорового питания!
中式早餐喝粥，非常___胃。
Китайский завтрак — каша, очень ___ желудок.
✅ 暖胃 = согревает желудок. 暖 = тёплый/согревать. Горячая каша 粥 = идеальный завтрак для желудка!
喝了鲜榨橙汁感觉很___。
После свежевыжатого сока чувствуешь себя очень ___.
✅ 清爽 (qīngshuǎng) = освежающий, бодрящий. Свежий сок = освежает. 很上头 — про крепкий кофе/алкоголь.
我早上___吃早饭___行。
Утром мне ___ завтракать ___ нельзя.
✅ 不…不行 = без… никак нельзя. 我早上不吃早饭不行 = Утром без завтрака мне никак.
一定要好好___早餐。
Обязательно нужно хорошо ___.
✅ 吃 (chī) = есть. 好好吃 = хорошенько поесть. 一定要 = обязательно нужно.

4 Правда или нет?

Соответствует ли утверждение материалу урока?

「稀饭」和「粥」— это разные блюда, ничего общего
✅ Неверно! 稀饭 и 粥 — это одно и то же: рисовая каша/кисель. 喝粥 = кушать кашу. Оба слова означают congee.
「培根」— это бекон, а「猪肉香肠」— это свиная сосиска. Это разные продукты.
✅ Верно! 培根 (péigēn) = бекон, 猪肉香肠 = свиная сосиска. Оба входят в английский завтрак 英式早餐.
「水煮蛋」готовится на сковороде с маслом
✅ Неверно! 水煮 = варить в воде. 水 = вода, 煮 = варить. Это варёное яйцо. На сковороде готовят 煎蛋.
「减脂早餐少油少盐超健康」— завтрак для похудения: мало масла, мало соли, очень полезно
✅ Верно! Это предложение прямо из урока. 少 = мало, 超 = очень/супер, 健康 = здоровый.
「拿铁」和「卡布」都是黑咖啡 — латте и капучино — это оба чёрный кофе
✅ Неверно! 拿铁 = латте (с молоком), 卡布 = капучино (с молоком и пеной). Чёрный кофе = 黑咖啡, эспрессо = 浓缩咖啡.
「没时间吃早饭」— значит у человека есть время на завтрак
✅ Неверно! 没时间 = нет времени. 没 = нет (для глаголов/существ.). Полностью: «нет времени есть завтрак».

5 Вставьте пропущенный иероглиф

Впишите иероглифы или пиньинь, затем откройте ответ.

1. 不吃早饭对___不好。 Не завтракать вредно для желудка.
胃 (wèi) — желудокЖелудок = 胃. Ср.: 心脏 = сердце, 肝 = печень, 肺 = лёгкие.
2. 赶时间就做快___早餐。 Когда торопишься, делай быстрый завтрак.
手 (shǒu) — рука快手 (kuàishǒu) = быстрый / ловкий (буквально «быстрые руки»). 快手早餐 = быстрый завтрак.
3. 鸡蛋很有___养。 Яйца очень питательны.
营 (yíng)营养 (yíngyǎng) = питательная ценность / питателен. Часто слышно: 这个很有营养！
4. 减脂早餐少___少盐超健康。 Завтрак для похудения: мало масла, мало соли — очень полезно.
油 (yóu) — масло / жир少油少盐 = мало масла, мало соли. 油 = масло. Ср.: 油条 тоже содержит 油!
5. 焦一点的烤面包___别香。 Слегка подгоревший тост особенно ароматный.
特 (tè)特别 (tèbié) = особенно. 特别香 = особенно ароматный / вкусный. 焦 = поджаренный/подгоревший.
6. 喝完感觉清___，很舒服。 После того как выпил — ощущение свежести, очень приятно.
爽 (shuǎng) — свежий/бодрый清爽 (qīngshuǎng) = освежающий, бодрящий. Говорят о свежем соке, прохладном напитке.
7. 按___吃，一日三餐。 Есть по расписанию, три раза в день.
时 (shí) — время按时 (àn shí) = вовремя, по расписанию. 按 = согласно/по, 时 = время.
8. 豆浆和油条是经典的___式早餐搭配。 Соевое молоко и жареные палочки — классика китайского завтрака.
中 (zhōng) — Китай / средний中式 = китайский стиль. Ср.: 英式 = английский, 西式 = западный, 欧式 = европейский.

6 Переведите на китайский

Мысленно переведите — затем откройте ответ с пиньинем.

1. Не есть завтрак вредно для желудка.
不吃早饭对胃不好。
Bù chī zǎofàn duì wèi bù hǎo.对 = для/по отношению к. Конструкция: 对…不好 = вредно для…
2. Утром мне без завтрака никак нельзя.
我早上不吃早饭不行。
Wǒ zǎoshang bù chī zǎofàn bù xíng.不…不行 = без… никак нельзя. 不行 = не годится / нельзя.
3. Когда торопишься, делай быстрый завтрак.
赶时间就做快手早餐。
Gǎn shíjiān jiù zuò kuàishǒu zǎocān.赶时间 = торопиться, 就 = тогда/то, 做 = делать.
4. Яйца очень питательны.
鸡蛋很有营养。
Jīdàn hěn yǒu yíngyǎng.很有营养 = очень питателен. Конструкция: 很有 + существительное = очень богат чем-то.
5. Три раза в день, нужно есть вовремя.
一日三餐，按时吃。
Yī rì sāncān, àn shí chī.一日三餐 = три приёма пищи в день (букв.: один день три еды), 按时 = по расписанию/вовремя.
6. Завтрак для похудения: мало масла, мало соли — очень полезно.
减脂早餐少油少盐超健康。
Jiǎnzhī zǎocān shǎo yóu shǎo yán chāo jiànkāng.超 = очень/супер (разг.). 少…少… = мало…, мало… Параллельная конструкция.

加油！🥣 Завтракайте вовремя — 一日三餐，按时吃！
